# Alternative cluster specifications for the criminology panel

A named list of three alternative group-assignment vectors for the
journals in `criminology_panel`. Used to test sensitivity of the
fragmentation ratio to cluster specification.

## Usage

``` r
criminology_alternatives
```

## Format

Named list with three elements, each a named character vector:

- `merge_cd_to_quant`:

  Reassigns *Crime & Delinquency* from the applied/practice cluster to
  the quantitative/empirical cluster.

- `merge_ps_to_applied`:

  Reassigns *Punishment & Society* from the critical/theoretical cluster
  to the applied/practice cluster.

- `merge_cjc_to_applied`:

  Reassigns *Criminology & Criminal Justice* from the
  international/comparative cluster to the applied/practice cluster.
